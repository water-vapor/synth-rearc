# c62e2108

The summary hints were broadly useful, especially the Opus note that horizontal blue guides trigger vertical repetition and vertical blue guides trigger horizontal repetition.

The official examples add one important detail: orthogonal guides act independently. A frame with both row-wise and column-wise guides produces a vertical strip and a horizontal strip, but it does not fill the corner quadrants via a Cartesian product of shifts.

Generator choices:
- One shared non-blue color per grid, matching the official task family.
- One or two hollow `4x4` frames.
- Each frame gets `1-3` blue edge guides chosen from top, bottom, left, and right.
- No extra distractor objects; spacing is enforced on both the input footprints and the eventual output strips so separate motifs stay visually distinct.
