The `arc2_opus46_summary.json` hint is consistent with the official examples and matches the actual rule: each 2-cell vertical segment expands into a full column, and each 2-cell horizontal segment expands into a full row.

The `arc2_sonnet45_summary.jsonl` hint was discarded as the primary explanation. Its repeated "template row" description is only a byproduct of overlaying the column stripes onto rows that are not themselves overwritten by horizontal stripes, so it obscures the real object-wise transformation.
