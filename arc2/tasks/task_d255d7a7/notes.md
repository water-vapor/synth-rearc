The Opus summary was directionally useful but incomplete, so I treated it as a partial hint rather than the full rule.

What actually moves is only the 3-cell-thick end-cap inside each long zero shaft. Any extra `9` markers outside that 3x3 cap, even when they are directly connected to the same non-7 component, stay fixed in place. The third training example shows this clearly: the lone `9` above the right-hand cap does not move, while only the 3x3 cap translates to the opposite border.
